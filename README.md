## Uso

### 1. Instala Hugo
[Instala Hugo](https://gohugo.io/overview/installing/)

### 2. Clona este repositorio
```
git clone git@github.com:fjvalles/fj-cabins.git
```

### 3. Inicia el servidor
```
hugo server
```
